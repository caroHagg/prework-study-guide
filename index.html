HTML

  • Definition:
    • One of the three cornerstones of web development
    • It provides content and basic framework for all pages of the webpage
    • It's made up of Elements which work together to build the foundation of the webpage

  • Different types of Elements can include Text elements, Image elements, Hearder elements, List elements:
    • The !DOCTYPE element defines what document type should be expected
    • The lang element describes the language to be expected
    • The head element contains information about the webpage.
    • The meta element defines the information about the webpage used only by the browser such as character set, author, etc.
    • The style element defines the CSS
    • The base element defines the base URL
    • The title element defines the tittle of the webpage
    • The body element represents the visible content shown to the user

CSS

  • There are three ways to style a webpage with CSS: inline, internal or external CSS
  • CSS rule is composed by a selector and a declaration block
  • A selector defines the element or attribute to which the rules will apply
  • Declarations contain two components: Property and Value
  • The Box Model is used to define space inside and outside elements, margina and padding
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Some common comands for Git
    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
    • git add -A: adds all changes to the current branch we are in
    • git pull: receive a branch's modifications into the local enviroment. Its best practice to have the lastest version on your computer
    • git push: add local changes to the remote github branch

JavaScript

  • JavaScript is a programming language that adds interactivity to youe webpage
  • Invented by Brendan Eich
  • It is versatile and beginner friendly
  • Control flow is the order in which a computer executes code in a script.
  • A variable is a named container that allows us to store data in our code.
  • There are different types of data such as string,number,boolewan, array, object
  • Conditionals are code structures that test is an expression returns true or false
  • Functions are a block of code that performe a specific task and are giving a unique name
  • The console is the test environment in which developers explore the code of the webpage
